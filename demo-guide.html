doctor.html?v=3023d4

最後に管理画面へ戻り、今日の受付・診療状況に反映されていることを見せます。 0件で止まらないよう、営業デモ用の山田家3名も予備表示されます。
owner.html?v=3023g
話短い営業トーク
この仕組みは、LINE公式を単なるお知らせ配信で終わらせず、診察券・受付・先生の確認・診療後フォローまでつなげる仕組みです。 患者さんは紙の診察券を探さずにLINEから出せます。 受付はQRを確認して受付済みにできます。 先生は今日やることと診療後のLINE文面を1画面で確認できます。 さらに管理画面で、今日の受付・診療状況も確認できます。 特にお子さんがいるご家庭では、お母さんのLINEに家族分の診察券をまとめられるので、医院側にも患者さん側にも分かりやすいです。

✓
個人LINE名や本番情報が出ていないか
営業前に、画面上の名前・LINE表示・URLを確認してください。
うまく反映されない時は、URLの最後に ?v=3023i のような数字を付けて開くか、Ctrl + F5 で強制更新してください。 GitHub Pagesは反映まで1〜2分かかることがあります。
